# RainbowKit Configuration

All RainbowKit configuration is centralized in the `app/config/` directory.

## Configuration Files

- `rainbowkit.ts` - Main RainbowKit configuration
- `wagmi.ts` - Wagmi configuration factory
- `theme.ts` - Theme configuration
- `chains.ts` - Supported chains
- `connectors.ts` - Wallet connector configuration
- `networks.ts` - Network and RPC configuration

## Environment Variables

See `.env.rainbowkit` for environment variable setup.
